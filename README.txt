Disable Post Edit (1.1.1) for MyBB 1.8
Created by: Starpaul20
Copyright: ©2017
License: GPL

Allows you to disable the ability to edit individual posts by the post author.

Editing can be disabled by checking the 'Disable Edit' box on the full edit page. Disabling editing does not effect moderator editing permission. This plugin offers full language support.


## INSTALLATION ##

1. Upload all files above, keeping the file structure intact.
2. Go to Configuration > Plugins
3. Click "Install & Activate"
4. Enjoy!


## UPDATING ##

If you're updating from any previous version, you must first deactivate the plugin, upload all new files and reactivate.
